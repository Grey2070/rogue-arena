📜 LOGS DE COMBAT

COMMENT JOUER

PICK/BAN : 1 ban + 1 pick chacun (15s).
DÉPLACEMENT : Tape une case bleue (3 PM/tour).
SORTS : Sélectionne un sort → les cibles s'illuminent → tape la cible.
PORTÉE : Chaque sort a une portée stricte en cases.
LDV : Certains sorts nécessitent une ligne de vue.
COMBOS : Touchez une cible déjà affectée pour déclencher un effet :
 🔥+❄ Feu+Glace → Explosion +2 dégâts + Gel
 ☠+⚡ Poison+Éclair → Paralysie +1
 🔥+🩸 Feu+Saignement → Brûlure Profonde +1
 ❄+🩸 Glace+Saignement → Gel Saignant
 💀+☠ Malédiction+Poison → Corruption +2
 🌑+☠ Ombre+Poison → Venin des Ombres +2
 🌱+🔥 Terre+Feu → Magma +1
 🌪+⚡ Vent+Éclair → Tempête +2
 ⚡+🩸 Éclair+Saignement → Paralysie Sanglante +1+Stun
💧 EAU :
 💧+🔥 Eau+Feu → Vapeur (Ralentissement)
 💧+❄ Eau+Glace → Gel Profond +1+Gel
 💧+⚡ Eau+Éclair → Court-Circuit +2+Stun
 💧+☠ Eau+Poison → Venin Dilué +1+Affaibli
 💧+🌱 Eau+Terre → Boue +1
 💧+🩸 Eau+Saignement → Hémorragie +2
✨ SACRÉ :
 ✨+💧 Sacré+Eau → Bénédiction (Soigne +2 le lanceur)
 ✨+☠ Sacré+Malédiction → Exorcisme +2+Cleanse
 ✨+🌑 Sacré+Ombre → Jugement +2+Affaibli
 ✨+⚡ Sacré+Éclair → Foudre Sacrée +1+Stun
 ✨+🔥 Sacré+Feu → Feu Divin +1
 ✨+🩸 Sacré+Saignement → Soin Rédempteur (Régén.)
